马克飞象-flawchart语法

发表于 2016-06-25   |   分类于 闻叨

写这篇分享的初衷

印象笔记方便保存，全平台上都可以查看，Markdown方便写作，格式美观，那怎么样能够结合二者的优点呢？这了推荐一款与印象笔记无缝连接的软件——马克飞象。

马克飞象的语法与一般的Markdown保持一致，但是它也有自己特色的地方，就是加入了流程图（flawchart）和时序图的语法，通过特定的代码块就可以在Markdown生成的页面中显示流程图和时序图啦。今天分享一下自己对流程图的语法的总结。

正式介绍Flawchart的语法

流程图的两个部分

流程图说到底，是由两个部分组成的：

  1. 含有文字的图形
  2. 图形之间的箭头连接线

图形

在马克飞象中，要生成含有文字的图形，先要确定你要用哪种图形，是要用椭圆形的“开始，结束”图形呢，还是用矩形的“过程”图形等等。语法大概如下：

  • 定义元素的语法

接下来具体讲下语法中的三个部分：

  1. tag
  2. type
  3. content

tag

tag其实就是我们给这个图形起的名字，自己随意指定，在第二段的时候连接元素用

type

type就是我们决定用什么图形，要注意的是type后面的冒号与content之间一定要有个空格

(type)类型有6种：

  1. start
  2. end
  3. operation
  4. subtoutine
  5. condition
  6. inputoutput

它们的形状分别为：

  1. 椭圆形
  2. 椭圆形
  3. 矩形
  4. 矩形里有两条竖线
  5. 菱形
  6. 平行四边形

content

content就是要在流程图的图形中显示的内容，中英文都可以
