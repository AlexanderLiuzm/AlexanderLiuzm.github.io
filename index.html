数组

把单个的数字组合在一起，分为静态数组和动态数组两种。

静态数组

一般在事先知道数组中有多少个数字的情况下会选择使用静态数组。

动态数组

一般在事先不知道数组中会有多少个数字的情况下回选择使用动态数组。

声明

静态数组

对静态数组的声明方式有两种：

  1. Dim Arr(10) As Integer
    括号中的10代表最大下标是10，即数组中最大的数字编号为10。VBA中的数组默认从0开始编号，所以上面这个数组中的数组编号从0~10，总共有11个数字。
  2. Dim Arr(1 to 10) As Integer
    这种申明方式同时定义了最小编号和最大编号，这个数组中只有10个数字。

动态数组

对动态数组的声明也有两种方式：

  1. Dim Arr()这种方式申明的数组是范写，适用范围更广
  2. Dim Arr(1 to S)其中的S是一个变量

变量

变量来源于数学，是计算机语言中能储存计算结果或能表示值抽象概念。变量可以通过变量名访问。在指令式语言中，变量通常是可变的；但在纯函数式语言（如Haskell）中，变量可能是不可变（immutable）的。——百度百科

上面的定义比较生涩拗口，我觉得 方骥老师 的比喻更加生动贴切。

这里的变量就像方程式里的 x, y，一开始你列了方程，但是暂时还没法算出一个数，之后只要把x变成具体的数字就可以算出一个值了。

过程

网上关于过程的定义：过程组被作为一个整体，指示Excel中如何执行特定任务执行的语句。执行的任务可以非常简单或非常复杂的，这是一个很好的做法，把程序分成较小的部分。

模块

网上对模块的定义：模块是其中代码被写入的区域。

VBA编辑窗口中的左侧有一个工程栏，里面显示的都是工程项目，在列表中是一个个模块。

变量

为什么要使用变量

运用变量来编写带有变动参数的语句代码，在为这些变量赋值之后就可以具体执行这些语句。

常用变量类型

    Excel中的Integer的范围是从负的30,000+到正的30,000+，并不包含所有的整数。因此如果要涉及到绝对值更大的数值，就要使用Long类型

不同级别的变量

过程

最常见的在子程序的编写过程中申明的变量就是过程级别的变量。过程级别的变量只有在当前自己编写的这个Sub中会使用，而不会和其他的Sub共享。

模块

在模块中申明的变量可以在不同的过程之间传递。简单的来说就是，在一个编辑窗口的最顶部（所有的Sub之上）申明的变量就是模块级别的变量。

公共

模块列表中有不同的文件夹，其中在Excel对象文件夹下的模块和在模块文件夹下的模块是有区别的，后者是全局类型的模块，在后者编辑窗口中可以申明公共变量。公共变量是全局变量，可以在不同的模块之间传递。

过程

ExcelVBA的过程有三种：

  1. 函数Function
  2. 子程序Sub
  3. 属性Property

这三种过程里最后一种不太常见，前面两种有一个明显的不同之处：

  • Function函数除了执行特定的语句外还会返回一个特定的结果
  • Sub则只是执行语句，不会返回一个结果

ExcelVBA中对象的属性和方法

发表于 2016-07-10   |   分类于 闻叨

什么是属性和方法

上面一篇文章中讲到了对象，简单的概括就是我们在Excel中对什么东西进行操作，这个什么东西就是所谓的对象。
现在讲到的属性和方法我们也可以打个比方。假如我们现在就谈论水果，那我们知道人会有品种；价格；口感等等标签来帮助我们区分不同的水果，这就是属性。而如果我们想要对它进行一些动作，比如洗水果;切开水果；吃水果等等的时候，这些词就是我们讲到的方法。
概括的来说：

  • 属性就是一个对象的一些标签
  • 方法就是我们要对对象进行的动作（或操作）

如何知道一个对象有哪些属性和方法

从上面的文章我们可以知道Excel中有很多的对象，这些对象的属性和方法又不近相同，怎么样才能知道一个对象有哪些属性和方法呢？难道我们要上网去找相应的表格，然后一个个背下来或者对着去找吗？其实巨硬公司早就想到了这一点，当我们进入到VBA编辑界面的时候，我们可以试着在一个对象的.后面单击右键，选择“属性/方法列表”，之后就可以看到一条下拉菜单，里面就是当前这个对象的属性和方法了。其中图标像一块被丢出去的绿色砖头的就是方法，而图标像一只手指着一个列表的就是属性，一动一静，还算好理解。

需要注意的是，当我们的对象是Selection的时候是无法出现方法/属性列表的，因为Selection是泛指被选择的对象，VBA还没有智能到能够自己判断之前到底选择了什么，所以无法给出对应的列表。

Excel VBA 对象

发表于 2016-07-10   |   分类于 闻叨

对象

是什么

所谓对象就是我们要对什么东西进行操作。例如，我们要对单元格进行填充上色，又例如我们复制当前一个工作表等等。
之前完成的VBA里我们仅仅在VBA的编辑窗口里改动了第一行和最后一行，给整个代码块套上了一个循环结构，但是中间的代码究竟代表什么我们没有去深究。
当我们理解了对象之后，我们就可以读懂中间的代码究竟对哪些东西进行了操作和修改。

具体分类


主要有以下几种类型：

  • Application
  • Workbooks
  • Sheets
  • Worksheets
  • Charts
  • Range

Application

Application指的是整个运行程序，当我们打开一个或者几个Excel表格文件的时候，这里的application指的就是Excel。

Workbooks

WorkBooks指的是我们在Excel表格中创建的工作表的合集。

Sheets

Sheets这个单词可能比起前几个会看起来比较眼熟，因为我们每次新建一个Excel的时候，左下角的页面标签就是Sheets1。Sheets就是工作表，与前面的工作簿有一字之差。就跟日常生活中一样，“表”是一张一张的，当很多张“表”被钉在一块，就成了一本“簿”，我们也可以这样来理解 Sheets 和 WorkBooks 的区别。

Worksheets

这里又出现了一个Worksheets，字面意思是工作表。那是不是我们选中的表就叫工作表呢？并不是的，其实Excel中有很多种的表格类型，当我们右键单击标签，选择插入时，会出先很多种表格类型让我们选择。

在图中我们可以看到除了“工作表”以外，还有“图表”和“宏表”等很多类型的表。我们平常用的最多的就是“工作表”，这里的Worksheets也特指“工作表”这一类型的表格。而上面提到的Sheets则泛指所有类型的表。

Charts

原理与上面的Worksheets差不多，Charts就是特指“图表”类型的表。

Range

Range就是单元格的意思，比起上面的几个词来说应该是比较好理解的了。
