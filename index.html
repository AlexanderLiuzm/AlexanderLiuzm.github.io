马克飞象-flawchart语法

发表于 2016-06-25

写这篇分享的初衷

很多人都用印象笔记，但是在我看来印象笔记自身的编辑功能是相对较差的，感觉使用起来对用户并不友好。这里要推荐一种跟平常使用不太相同的编辑方法——Markdown。本人私以为Markdown编辑相比起富文本来说是比较快捷的，只用在标题文本前面加一个简单的“#”，就可以配置好标题的样式，更多的Markdown语法可以点击这里。不过Markdown要有一定的学习期，熟练之后对编辑的速度才有

最重要的语法框架

  • 定义元素的语法

  • 连接元素的语法

分解框架

定义

tag可以自己指定，在第二段的时候连接元素用

(type)类型有6种：

  1. start
  2. end
  3. operation
  4. subtoutine
  5. condition
  6. inputoutput

content就是要在流程图的框中显示的内容，中英文都可以

连接

直接用->来连接之前定义好的两个元素

上面举的condition类型是特殊类型，要写两行，yes情况写一行，no情况写一行

默认情况箭头都是往下，如果想让流程图往右或左写就要在括号中注明(right)或者(left)
